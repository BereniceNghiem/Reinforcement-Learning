# Reinforcement Learning

This is the git repository of the class [Reinforcement Learning](https://moodle.ip-paris.fr/course/view.php?id=15722) at Institut Polytechnique de Paris.
It contains Python scripts and notebooks for the labs. 

The repository is updated during the class. Please make sure that you get the last version for each lab.

## Getting started

```
python3 -m venv .env
source .env/bin/activate
pip install -r requirements.txt
```
